

# Site Web - FC Barcelona

Ce projet a été réalisé dans le cadre de la SAE 14 L'objectif était de concevoir et développer un site web complet et responsive.

J'ai choisi le thème du FC Barcelone pour présenter l'effectif 2025, le calendrier des matchs et l'histoire du club.

## Accès au site

Le site est hébergé en ligne via GitHub Pages.
[Cliquez ici pour visiter le site](https://ahmed2720.github.io/sae/)

## Présentation du projet

Le site est composé de trois pages principales :

1.  Accueil (index.html) :
    Présentation de l'équipe complète, mise en avant d'un joueur (Lamine Yamal) et palmarès du club. Une animation de chargement a été ajoutée.

2.  Calendrier (calendrier.html) :
    Affichage des prochains matchs de la saison. J'ai utilisé une mise en page en grille pour structurer les dates et les adversaires et j'ai ajoutée le classement de La Liga et un apercu de la forme de l'équipe sur les derniers matchs.
PS: La page par exemple calendrier ne se met pas a jour automatiquement car je ne savais pas comment le mettre en place ducoup c'est le calendrier , le classement et la forme au moment ou j'ai code la page.

3.  À Propos (apropos.html) :
    Présentation du contexte du projet, explication du choix du thème.

## Technologies utilisées


  - HTML5 : Pour la structure sémantique des pages (header, main, section, aside, footer).
  - CSS3 : Pour le design.

J'ai utilisé plusieurs outils:

  - Flexbox : Pour la structure globale et le menu.
  - CSS Grid : Pour aligner le calendrier des matchs.
  - Media Queries : Pour le responsive design. Sur mobile le site s'adapte et les colonnes s'empilent pour rester lisibles.
  - Animations CSS : Pour le loader au démarrage et les survols des menu en haut a gauche.
